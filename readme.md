**VizAI** is a Streamlit-based AI-powered data analysis and visualization tool developed by Ethicallogix. It allows users to clean, preprocess, visualize, and analyze datasets effortlessly. The platform also includes secure login, registration, and Google OAuth authentication.

---

## Features

### 1. **User Authentication**
- Username/password login
- Google OAuth login
- Default admin creation (`hasi`) for initial setup
- Secure password hashing

### 2. **Data Analysis & Cleaning**
- Load and preview datasets
- Handle missing values
- Delete or fill columns/rows
- Data standardization and normalization
- Label encoding, one-hot encoding, and ordinal encoding

### 3. **Visualization**
- Interactive charts with Plotly (bar, line, scatter, pie, etc.)
- Virtualization of datasets for better readability
- Summary statistics and insights generation

### 4. **Google Generative AI Integration**
- AI-powered suggestions for data analysis
- Text processing and pattern detection

### 5. **Multi-page Streamlit App**
- Clean multi-page navigation (Home, Admin, Data Page, Virtualization, Syntax, etc.)
- Customizable page layout with hidden sidebar and menu for a polished look

---

## Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **Data Processing:** Pandas, Numpy, Scikit-learn
- **Visualization:** Plotly
- **Authentication:** Google OAuth + hashed credentials
- **AI Integration:** Google Generative AI
- **Environment Management:** Python-dotenv
